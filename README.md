## What is SSAFY?

![SSAFY](./img/ssafy.png)

삼성 청년 SW 아카데미(SSAFY)란 삼성의 SW 교육 경험과
고용노동부의 취업지원 노하우를 바탕으로 취업 준비생에게 SW 역량 향상 교육 및
다양한 취업지원 서비스를 제공하여 취업에 성공하도록 돕는 프로그램입니다.
